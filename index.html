How to Get Raw URL
GitHub Gist
1. Go to gist.github.com
2. Paste your script
3. Click Create
4. Click Raw button
5. Copy the URL
Pastebin
1. Go to pastebin.com
2. Paste your script
3. Click Create
4. Click Raw button
5. Copy the URL
GitHub Repository
1. Upload lua file to your repo
2. Click on the file
3. Click Raw button
4. Copy the URL
How to Use

Methods Explained

Encoded: Uses base64. Good for most scripts.

Byte: Uses byte numbers. Harder to read.

Hex: Uses hex values. Very hard to read.

Raw URL: Uses external hosting. Best for large scripts.
